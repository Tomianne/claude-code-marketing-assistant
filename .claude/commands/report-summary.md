Summarise the marketing data or report the user has shared.

## Step 1: Identify what was shared
The user will paste data or a report into their message. Accept any format: numbers, a paragraph summary, campaign results, email metrics.

If nothing was pasted, ask: "Please paste the report or data you want me to summarise."

## Step 2: Write the summary

Structure:
```
## Overview
[2-3 sentences. What period does this cover? What was the overall result?]

## What worked
[2-3 bullet points with specific evidence from the data.]

## What did not work
[2-3 bullet points with specific evidence. Be direct.]

## Key insights
[3-5 observations that explain the patterns -- not just what happened, but why and what it means.]

## Recommended actions
[3-5 specific, practical next steps.]
```

Rules:
- Only reference what is actually in the data -- no invented benchmarks
- If data is incomplete or comparisons cannot be made fairly, flag it
- Keep language plain -- this is for a decision-maker, not an analyst
- If the data is too vague to draw conclusions, say so and ask for more context

## Step 3: Output
If the summary is over 200 words, offer to save it to projects/strategy/YYYY-MM-DD-report-summary.md
Otherwise present it in the conversation directly.
